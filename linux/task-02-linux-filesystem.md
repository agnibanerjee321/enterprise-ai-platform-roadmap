# Task 2.1 — Linux Filesystem and Navigation

## Goal

Understand the Linux filesystem structure, navigation commands, and important directories used in platform engineering.

## Why this matters

Linux is the base of servers, containers, Kubernetes nodes, CI/CD runners, automation tools, and troubleshooting workflows.

## Important Linux Directories

| Directory | Purpose |
|---|---|
| / | Root of the Linux filesystem |
| /home | User home directories |
| /etc | System configuration files |
| /var | Variable data such as logs and cache |
| /var/log | System and application logs |
| /tmp | Temporary files |
| /usr/bin | Common Linux commands |
| /opt | Optional or enterprise software |
| /mnt | Mounted filesystems |
| /root | Root user's home directory |
